# 🦺 SafetyEye – AI Powered Workplace Occupancy & Safety Monitor

## Overview

SafetyEye is an AI-powered workplace safety monitoring system developed using YOLOv8, OpenCV, Python, and Streamlit.

The system automatically detects Personal Protective Equipment (PPE) compliance in real time using CCTV or webcam feeds. It identifies workers who are not wearing safety equipment such as helmets, safety vests, and masks.

## Features

* Real-Time PPE Detection
* Helmet Detection
* Safety Vest Detection
* Mask Detection
* Worker Occupancy Monitoring
* Violation Logging
* Screenshot Capture
* AI Risk Prediction
* Live Dashboard Analytics
* Multi-Camera Monitoring
* Safety Compliance Tracking
